# Portfólio (Frontend)

## Português

Este é o projeto do frontend do meu portfólio pessoal, concebido como uma demonstração prática do desenvolvimento de aplicações no mundo real. Todas as etapas e implementações feitas aqui foram escolhidas especificamente para demonstrar o passo a passo da elaboração de um projeto realista, de acordo com os desafios que encontramos no dia a dia do nosso trabalho.

### Destaques

- Planejamento do projeto (briefing, tarefas e conteúdo).  
- Desenvolvimento de UX/UI design.  
- Transformação do projeto visual em aplicação web com HTML e CSS.  
- Modelagem de banco de dados.  
- Construção de API REST.  
- Autenticação de usuários.  
- Criação de rotas públicas e privadas.  
- Desenvolvimento de dashboard CMS (Content Management System) para gerenciamento de conteúdo.  
- Gerenciamento avançado de formulários com campos dinâmicos (incluindo validação e máscara).  
- Gerenciamento de arquivos e diretórios (criação, remoção e upload).  
- Gerenciamento do contexto da aplicação (para controle de temas e traduções).  

## English

This is the frontend project of my personal portfolio, designed as a practical demonstration of real-world application development. All stages and implementations here were specifically chosen to showcase the step-by-step process of developing a realistic project, according to the challenges we encounter in our day-to-day work.

### Highlights

- Project planning (briefing, tasks, and content).  
- UX/UI design development.  
- Converting visual design into a web application using HTML and CSS.  
- Database modeling.  
- REST API development.  
- User authentication.  
- Creation of public and private routes.  
- CMS (Content Management System) dashboard development for content management.  
- Advanced form handling with dynamic fields (including validation and masking).  
- File and directory management (creation, deletion, and upload).  
- Application context management (for handling themes and translations).
